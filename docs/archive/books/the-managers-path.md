# Review of The Manager's Path
Book Link: [https://smile.amazon.com/Managers-Path-Leaders-Navigating-Growth/dp/1491973897/](https://smile.amazon.com/Managers-Path-Leaders-Navigating-Growth/dp/1491973897/)

I feel like I have already read this book given the amount of blog posts, talks, and other references to it I've seen over the last 8 years but I wanted to give it a listen to get the full set of information. I found it to be a great reference for what to expect through your career in software engineering, into management, and then to executive (assuming that is the path you want). I don't know that I learned anything new in particular given how much of this book has been shared through other mediums, but getting to follow the narrative and seeing how Camille intentionally builds and layers each role and its responsibilities was a very nice presentation. I would definitely recommend this book to anyone look at or in the software engineering -> manager -> executive leader career path and potentially couple the recommendation with [An Elegant Puzzle](an-elegant-puzzle.md) for a slightly different approach to the same problem.
